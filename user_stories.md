# Tic Tac Toe

The Command Line Games, Inc. edition

```
01. As a player,
So I can play Tic Tac Toe,
I need a square board on which to play it.

02. As a player,
So I can play Tic Tac Toe,
I need a set of rules I can play by.

03. As a player,
So I can always play Tic Tac Toe,
I need to play against either a human or the computer.

04. As a player,
So I enjoy playing Tic Tac Toe,
I need the computer to play at multiple difficulty levels.

05. As a player,
So I enjoy playing Tic Tac Toe,
I want the game to gracefully handle the bad input I provide by mistake.

06. As a player,
So I know what is going on while playing Tic Tac Toe,
I want the game to give me clear and relevant user messages.

07. As a player,
So I enjoy playing Tic Tac Toe,
I want to be able to choose the game type (human/human, human/computer, computer/computer).

08. As a player,
So I enjoy playing Tic Tac Toe,
I want to be able to choose which player goes first.

09. As a player,
So I enjoy playing Tic Tac Toe,
I want to be able to choose the sign I will use to play.

10. As a player,
So I can play Tic Tac Toe at any time and on various devices,
I want to be able to play it via multiple user interfaces (CLI and web).

```
